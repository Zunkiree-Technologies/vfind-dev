# V-Find Workflow Patterns

Project-specific development patterns.

---

## Development Workflow

### Starting Development
```bash
cd C:\Users\sadin\OneDrive\Desktop\Simplifycodes\DevFolder\vfind-dev
npm run dev
```

### Before Coding
1. Run `/sync` to load context
2. Check `status-current.md` for latest state
3. Review `next-session.md` for priorities

### After Session
1. Run `/memorize` to save progress
2. Update status-current.md
3. Log significant work in progress-journal.md

---

## Code Patterns

### Component Structure
- Functional components with TypeScript
- React Hook Form for form handling
- Zod for validation schemas
- Radix UI primitives for accessible components

### State Management
- React Context for global state
- Custom hooks for reusable logic

### Styling
- Tailwind CSS utility classes
- tw-animate-css for animations
- next-themes for dark mode
